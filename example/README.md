# flutter_just_toast_example

Demonstrates how to use the flutter_just_toast plugin.

```
Toast.show(message:"Hello world",duration: Delay.SHORT);
````

Duration :
```
Duration.SHORT,
Duration.LONG
```

#### Customized Toast

```
Toast.show(message:"Hello world",duration: Delay.SHORT,
backgroundColor: Colors.blue,
textColor: Colors.black);

```
