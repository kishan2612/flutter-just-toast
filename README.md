# flutter_just_toast

A minimal flutter toast plugin

## Usage 

In pubspec.yaml

```
flutter_just_toast: ^0.0.1

```

Import flutter_just_toast.dart

```
Toast.show(message:"Hello world",duration: Duration.SHORT);
````

Duration :
```
Duration.SHORT,
Duration.LONG
```

IOS Interdependency :
[Toaster](https://github.com/devxoul/Toaster)
